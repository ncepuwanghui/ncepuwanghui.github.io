4. 遇到的问题

4.1 Algolia搜索跳转时显示undefined

参考链接：https://github.com/iissnan/hexo-theme-next/issues/1084

方法一：在站点下node_modules/hexo-algolia/lib/command.js文件中添加path，然后执行hexo algolia重新生成索引。如下：

方法二：重新安装插件hexo-algolia，重装时指定版本为0.2.0，在站点根目录下执行npm install hexo-algolia@0.2.0，然后重新生成索引hexo algolia。

4.2 Algolia搜索第二次点击时出错

参考链接：https://github.com/iissnan/hexo-theme-next/issues/1354

4.3 重装插件时提示权限错误

参考链接：https://github.com/nodeschool/discussions/issues/305

安装插件hexo-algolia时，出现错误，如下：

解决方法： 经检查发现，～/.npm目录下目录文件所有者和组均为root，修改.npm目录所有者为当前普通用户。

若还有错误，可发现站点根目录下node_modules/.bin的所有者和组也为root，将其修改为当前普通用户。
